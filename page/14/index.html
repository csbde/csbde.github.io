emacs中中文显示的时候一个汉字只显示半个的问题

发表于 2008-12-05 | 分类于 SoftWare

这是一个困扰了我大半年的问题:下载emacs以后显示汉字老是经常出现半个汉字的情况.一直以为是配置问题,但实验了几个网上无数人都测试成功的配置文件,我也还是会出现半个汉字的现象,期间在办公室和家里的电脑都有测试过,都不能正常显示汉字,期间重装过几次系统,同样不能解决此问题,只好继续研究VIM.

终于在2008年12月5日这天,在又耗费了N小时后看到了网上某人的一句话,豁然开窍,成功安装运行emacs并正常显示汉字
都是灵格斯(lingoes)它在安装时在c:windowsfonts下安装了它自己的字库(lingoes.ttf),删除即可. 而我每次重装系统都是首先安装lingoes.万恶的灵格斯(虽然很好用,但有必要装它自己的字库吗?),半年的时间都浪费在不断的试验上了.而且我怀疑我办公室的电脑在运行ooo时有时会出现重字的现象,也跟这个有关,明天去办公室把这个删掉试验下

Firefox 中的Google 搜索被流氓的四川电信DNS劫持以后的解决方法

发表于 2008-12-05 | 分类于 InterNet

四川电信拦截用户的搜索引擎，在网民提交搜索词汇和搜索引擎返回搜索结果的过程中插入 cninspirit 这个流氓网站。神不知鬼不觉，用户就帮电信挣了几美分。而且最难以忍受的是，我设置的搜索引擎是google.com，但它提交的结果是google.cn的，并且经常只提交一部分搜索词，让搜索结果完全不可用。以电信的用户数量，搜索引擎劫持一天就能挣上不少钱。
经N次实验得出的结果是，只要是firefox提交的搜索就会被劫持。很悲哀的发现不管用什么方法都不能组止被劫持，换DNS也不行。不得已，开始研究从Firefox搜索机制上下功夫，可惜功力不够深（小白一个）所以没有结果，但不小心在网上看到一个插件

Add to Search Bar 1.7

欣喜若狂，安装以后一切问题迎刃而解，HOHO，再也不用给垃圾的电信贡献广告点击了

点击安装以后在焦点处于网页的输入框的时候会在鼠标右键会添加一项add to searchbar,点击会让你输入搜索名字和图标等,填好以后确定就可以了.现在到搜索引擎栏里选择你刚添加的搜索引擎,试试吧,不会被劫持了,证明劫持者良心还没有坏透,只是劫持有特殊字串的提交结果

IE用户只需要更改你集成的搜索引擎的提交地址就可以了.下面给出几个常用的搜索引擎提交格式,以便参考

google.cn的

http://www.google.cn/search?q=XXXXXXXX

其中的XXXXXXX就是你要搜索的词语,比如你要搜索 “九王爷”,那么地址就应该是http://www.google.cn/search?q=九王爷

google的

http://www.google.com/search?q=XXXXXXXX

其中的XXXXXXX就是你要搜索的词语,比如你要搜索 “9wy.net”,那么地址就应该是http://www.google.com/search?q=9wy.net Google.com网址里面的中文一般需要进行编码,英文是可以直接带在后面的

百度的:

http://www.baidu.com/s?wd=XXXXXX

跟上面一样,如果是比如你要搜索 “九王爷”,那么地址就应该是http://www.baidu.com/s?wd=九王爷

Firefox我使用以来收集的一些优化方法

发表于 2008-11-24 | 分类于 SoftWare

在地址栏输入：about:config.

Firefox默认的历史记录太大,会让浏览器慢, 还有一些别的默认记录设置也过高,所以设置小点速度会小有提升.

browser.history_expire_days：表示历史纪录过期天数，默认是180天！ 7
browser.history_expire_days_min：是用户值，即在“Firefox首选项”、“隐私”、“历史”当中，表示“保存我的历 史，至少X天”。即使你在首选项中把“至少X天”设置成了了1天，或取消，Firefox还是会保存浏览历史，因为系统保留值为180天。 31
browser.history_expire_sites：这个是最大保存的站点数量，默认是四万个。 400
将browser.sessionhistory.max_total_viewers修改值为5或更小{页面快进/快退功能中保存的页面总数，默认是-1(无限)}
network.http.redirection-limit [整数]决定接受多少连续的重定向。比如说你进入一个站点旧的网址，可能会被重定向到一个新的网址，这叫一个重定向。建议5.一个网站重跳5次以上可真累

网络部分优化
network.http.pipelining”（按键“n”可以快速定位到network.），双击“network.http.pipelining”,将其值改成“true”（双击自动变为true）。
network.http.pipelining.maxrequests [整数]决定使用HTTP Pipelining特性时发送的最大连接请求。默认值为4，最大值为8，比8大的值会被忽略，1表示不使用HTTP Pipelining特性，建议将该值设置为8。
在Firefox窗口任意位置点击右键，选择“新建”—“整数”在弹出的对话框中，输入“nglayout.initialpaint.delay”并将其值改为“0”，表示当Firefox浏览器收到网站回复的信息后能够快速的反应。
还有重要的一步,把ipv6关了
network.dns.disableIPv6 true

Windows 下更改GTK显示字体

发表于 2008-11-05 | 分类于 SoftWare

前天将XP系统字体换成了雅黑，结果pidgin中文显示都是方框，猜想是这个雅黑字体闹的，google了一下，修改GTK+配置，使Windows XP下在GTK+ 中使用微软雅黑字体，然后就OK了。
如果一个 GTK+ 程序在 Windows 下运行，默认的中文字体是很难看的，下面就介绍如何在 GTK+ 中使用微软雅黑字体。

找到 …\etc\pango\pango.aliases 文件，添加一行：

msyh = “Microsoft YaHei”
这就声明了雅黑字体，当然 Windows 字体目录下面应该有雅黑字体，Microsoft YaHei 就是字体的名字，比如还有 Simsun 宋体。我们把雅黑字体定义为一个字体族 msyh。

找到 …\gtk-2.0\gtkrc文件，将下 面内容加到后面

style “user-font” {
font_name = “msyh 9”
}

widget_class “*” style “user-font”

gtk-font-name=”msyh 9”
这样应用程序的字体就变成微软雅黑了。

使用OpenDNS远离电信的114病毒广告网站

发表于 2008-10-27 | 分类于 Uncategorized

电信的114是个流氓，不管是输入错误还是没有找到这个站点它都会把域名劫持到114.vnet.cn或者别的以114开头的域名上面。而且还会重定向？或者是转发到一些个病毒网站上面。实在是对电信流氓忍无可忍。解决方法也非常简单，更改DNS解析服务器，只要不使用电信的DNS解析服务它就不会劫持了。

这里推荐使用OpenDNS，国外的。非常的实用。

如果你E文还不错，能够浏览E文页面的话，建议你到http://www.opendns.com 建立你自己的DNS规则。而如果你只是想远离114这个流氓的话就简单得多了，直接在控制面板->网络连接->你使用的连接中 点鼠标右键，选择TCP/IP，在DNS服务器里添208.67.222.222 和 208.67.220.220 确定就可以了
