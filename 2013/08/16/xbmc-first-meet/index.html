影音播放软件xbmc

发表于 2013-08-16 | 分类于 Uncategorized

在linux下面看在线影音始终是个问题，特别是pptv这些。 虽然在线也能播放，但是我还是想能窗口播放，并且能一边播放一边做事。
暂时来说，还没有找到符合心意的软件，原计划用python自己做一个。但还是怕麻烦，找了下，有个xbmc下面的插件，能看pptv，赶紧的装个xbmc试试。

  • 首先
    根据xbmc官方wiki 里的安装方法

    sudo apt-get install python-software-properties pkg-config
    sudo apt-get install software-properties-common
    sudo add-apt-repository ppa:team-xbmc/ppa
    sudo apt-get update
    sudo apt-get install xbmc
    试用
    试用有个小问题，在xbmc退出后，我的gnome3会黑屏或者闪动。根据我的经验，这应该是驱动问题。 现在我使用的还是ubuntu自带的开源的。 还是换成厂家自带的试试。
    打开系统设置，附加驱动，选中一个想要的驱动，激活。
    重启电脑一切都好了     安装插件
