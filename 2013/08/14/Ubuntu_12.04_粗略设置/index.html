## <a name = "shell" ></a> Shell 选择

就目前来说，Shell里面应该zsh是首选。
当然，要是觉得麻烦，系统自带的bash就足够使用了。
但bash对我来讲，就是补全不能满足我得要求，我希望的是能自动补全，而且最好能在apt这些时也能很好的补全。 所以选中了zsh
安装：

<pre>` apt-get install zsh
 wget --no-check-certificate https://github.com/robbyrussell/oh-my-zsh/raw/master/tools/install.sh -O - | sh
 cp ~/.oh-my-zsh/templates/zshrc.zsh-template ~/.zshrc
 chsh -s /bin/zsh
`</pre>

重启系统生效

## <a id = "xbmc"></a> XBMC 影音套装

用电脑的很大一个原因就是影音，在线/不在线的。不在线的我们可以用mplayer来播放，
音乐视屏都没有问题。但在线视屏就比较难了，不可能每个视屏都去找到地址让mplayer来播放。
那么，[xbmc](http://xbmc.org/ "官网") 就隆重登场了。
安装方法戳[这里](http://wiki.xbmc.org/index.php?title=Installing "Installing"),英文的
UBUNTU下安装：

<pre>`sudo apt-get install python-software-properties pkg-config
sudo apt-get install software-properties-common
sudo add-apt-repository ppa:team-xbmc/ppa
sudo apt-get update
sudo apt-get install xbmc

xbmc是跨平台的播放中心。能通过添加插件来让其的功能近乎无穷的扩展
跟linux下的软件差不多，能通过插件库增加功能，这里主要介绍三个插件库：

  • 官方插件库 这个没什么好说的，自带的。

  • 中文插件库

    是下载名字为： repository.googlecode.xbmc-addons-chinese-XX.zip 的包
    这个插件库有点小问题，反正在我的电脑上得手动打开 ~/.xbmc/addons/repository.googlecode.xbmc-addons-chinese/addon.xml
    将里面所有的http://都换成https才能正常的下载使用
    安装了这玩意后，打开xbmc进入到系统设置-扩展功能里面就能从相关库里面安装像 pps影音， pptv ， ku6 等等等在线影视,享受在线影视噢

  • 18插件库

    这个大家都懂的

  • HDPFANS的插件库

    有一些高清的视屏

PS: 默认安装的只能显示英文界面，需要安装字体

  1. 第一次进入xbmc是英文版的，需要做如下操作进入中文界面，进入system->appearance
  2. 选择skin，把fonts设置为arial based，这步很关键，不改fonts的话会导致中文字符无法显示。
  3. 到international里面设置lanuage为Chinese(simple)
  4. 系统语言设置好后，接下来就是安装插件了，进入 系统设置 -> 扩展功能
