使用方法

当你做完以上步骤的时候，你就可以无论在什么目录下在 VIM 里面通过下面的指令进行新建一篇日志了：

:HexoNew artical-name

以及下面的指令来打开一篇已存在的日志：

:HexoOpen artical-name
